## Overview

A high-performance, fully responsive developer portfolio built with React and Vite. Features a dark glassmorphism UI, scroll-triggered animations, lazy loading, and a clean professional layout.

## Run Locally

```bash
git clone https://github.com/faizan98653/my-portfolio.git
cd my-portfolio
npm install
npm run dev
```

Open `http://localhost:5173` in your browser.

## Deploy Changes

```bash
git add .
git commit -m "your update message"
git push
```

Vercel auto-deploys on every push. Live site updates in 1-2 minutes.
